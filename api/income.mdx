The income module implements income approach valuation methods including discounted cash flow (DCF) analysis, capitalization rate calculations, and internal rate of return (IRR) derivation for commercial real estate.

## Core Functions

### derive_prices

```python
derive_prices(
    target_irr: float,
    exit_cap_rate: float,
    entry_noi: float,
    noi_growth: float,
    holding_period: int
) -> tuple[float, float]
```

Calculate entry and exit prices using a discounted cash flow (DCF) model.

This function implements a standard DCF analysis where annual NOI cash flows are discounted at the target IRR, and the property is sold at the end of the holding period based on the exit cap rate.

<ParamField path="target_irr" type="float" required>
  Target internal rate of return as a decimal (e.g., 0.10 for 10%)
</ParamField>

<ParamField path="exit_cap_rate" type="float" required>
  Exit capitalization rate as a decimal (e.g., 0.06 for 6%)
</ParamField>

<ParamField path="entry_noi" type="float" required>
  Net operating income at purchase (NOI₀)
</ParamField>

<ParamField path="noi_growth" type="float" required>
  Expected annual NOI growth rate as a decimal (e.g., 0.03 for 3%)
</ParamField>

<ParamField path="holding_period" type="int" required>
  Holding period in years
</ParamField>

<ResponseField name="entry_price" type="float">
  The calculated purchase price based on DCF analysis
</ResponseField>

<ResponseField name="exit_price" type="float">
  The expected sale price at the end of the holding period
</ResponseField>

**DCF Equation:**
```
Entry Price = Σₜ₌₁ᴴ [NOI₀ × (1 + g)ᵗ / (1 + IRR)ᵗ]
            + [NOI₀ × (1 + g)ᴴ⁺¹ / exit_cap] / (1 + IRR)ᴴ

Exit Price = NOI₀ × (1 + g)ᴴ / exit_cap
```

### derive_irr

```python
derive_irr(
    entry_price: float,
    exit_price: float,
    entry_cap_rate: float,
    noi_growth: float,
    holding_period: int
) -> float
```

Calculate the implied internal rate of return given observed entry and exit prices.

This function uses a numerical solver to find the IRR that makes the net present value of all cash flows equal to zero.

<ParamField path="entry_price" type="float" required>
  Actual purchase price of the property
</ParamField>

<ParamField path="exit_price" type="float" required>
  Observed sale price at end of holding period
</ParamField>

<ParamField path="entry_cap_rate" type="float" required>
  Entry capitalization rate as a decimal (e.g., 0.06 for 6%)
</ParamField>

<ParamField path="noi_growth" type="float" required>
  Annual NOI growth rate as a decimal
</ParamField>

<ParamField path="holding_period" type="int" required>
  Holding period in years
</ParamField>

<ResponseField name="irr" type="float">
  The implied IRR as a decimal
</ResponseField>

**Method:** Uses Brent's method (`scipy.optimize.brentq`) to solve the DCF equation for IRR.

### calculate_cap_rate_growth

```python
calculate_cap_rate_growth(
    sale_price_growth: np.ndarray,
    noi_growth: np.ndarray
) -> np.ndarray
```

Calculate annual cap rate changes from sale price and NOI growth rates.

Given `NOI = Sale Price × Cap Rate`, the percentage change in cap rate is approximately:
```
ΔCap Rate ≈ (1 + ΔNOI) / (1 + ΔSale Price) - 1
```

<ParamField path="sale_price_growth" type="np.ndarray" required>
  Annual percentage changes in sale price as decimals (e.g., `[0.01, 0.015, 0.02]` for 1%, 1.5%, 2%)
</ParamField>

<ParamField path="noi_growth" type="np.ndarray" required>
  Annual percentage changes in NOI as decimals
</ParamField>

<ResponseField name="cap_rate_growth" type="np.ndarray">
  Annual percentage changes in cap rate as decimals
</ResponseField>

## Mathematical Background

### DCF Model Assumptions

1. **Cash Flows**: Annual NOI grows at constant rate `g`
2. **Discount Rate**: All cash flows discounted at IRR
3. **Exit Value**: Property sold at NOI/cap_rate formula
4. **Holding Period**: Integer number of years

### Net Present Value Equation

```
NPV = Σₜ₌₁ᴴ [NOIₜ / (1 + IRR)ᵗ] + [Exit Price / (1 + IRR)ᴴ] - Entry Price = 0
```

The functions use `scipy.optimize.brentq` for numerical root-finding when solving for IRR or NOI growth.
